# Chapter 7: Math Operators

You know how to use `+` to add numbers. But Python can do much more than just addition. 

### 1. The Problem

You need to calculate a 15% tip on a $114.15 meal, doing that math manually in your head defeats the whole purpose of programming! The computer should be doing the heavy lifting for you.

### 2. The Tool

Python has built-in math operators that work exactly like a standard calculator. 

* **Addition:** `+`
* **Subtraction:** `-`
* **Multiplication:** `*`
* **Division:** `/`
* **Modulo (Remainder):** `%` *(e.g., `10 % 3` prints `1`, because 10/3 has the remainder of 1)*

### 3. The Task

Build a calculator using `input` and mathmatical operators.
